Active learning¶

class activelearn.activelearner.active_learner.ActiveLearner(model, example_selector, labeler, batch_size, num_iters)¶

A class which allows to learn a given model by actively querying the labels of unlabeled instances using Pool-based active learning.

Parameters:
  • model (Model) – Scikit-Learn Model to learn
  • example_selector (ExampleSelector) – example selector to query informative examples
  • labeler (Labeler) – A labeler which fetches labels from the oracle/human
  • batch_size (number) – The number of examples to be labeled per iteration
  • num_iters (number) – Number of iterations to run the active learner

learn(unlabeled_dataset, seed, exclude_attrs=None, context=None, label_attr='label', not_sure_label_allowed=False)¶

Performs the Active Learning Loop to help learn the model by querying the labels of the instances

Parameters:
  • unlabeled_dataset (DataFrame) – A Dataframe containing unlabeled examples
  • seed (DataFrame) – A Dataframe containing initial labeled examples which is used to learn the initial model
  • exclude_attrs (list) – A list of attributes to be excluded while fitting the model (Defaults to None)
  • context (dictionary) – A dictionary containing all the necessary context for the labeling function
  • label_attr (string) – A string indicating the name of the label column in the labeled dataset. Defaults to label
Returns:

A learned model
