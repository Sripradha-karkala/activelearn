Commandline based labeler¶

class activelearn.labeler.cli_labeler.CliLabeler(get_instruction_fn, get_example_display_fn, labels)¶

A command line labeler for labeling raw instances

Parameters:
  • get_instruction_fn (Function) – User provided function which specifies the instruction to be displayed to the user before a particular example’s label is queried from the user.
  • get_example_display_fn (Function) –
    User provided function which
    specifies how to fetch the raw examples to be labeled. This function will take 2 arguments namely a feature vector and a context object. The context is an object provided by the user that contains the necessary data to obtain the raw representation of the feature vector to be labeled. This function will provide the necessary logic to obtain the raw representation of the examples to be labeled using the provided context.

    labels (Dictionary): A dictionary with the label text and values
